# 数据库设计文档 v1.0

> 数据库类型：SQLite
> 数据库文件：`data/movie_analysis.db`
> 字符编码：UTF-8

---

## 1. 数据库整体结构

```
movie_analysis.db
├── movies          → 电影主表（核心表）
├── reviews         → 短评表（可选/加分项）
├── box_office_log  → 票房日志表（趋势数据）
├── crawl_record    → 爬虫记录表
└── system_config   → 系统配置表
```

---

## 2. 表结构设计

### 2.1 movies — 电影主表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |
| title | TEXT | NOT NULL UNIQUE | 电影名称 |
| genre | TEXT | NOT NULL | 类型（逗号分隔，如"动作,科幻"）|
| director | TEXT | | 导演 |
| actors | TEXT | | 主演（前 5 位，逗号分隔）|
| release_date | TEXT | | 上映日期 (YYYY-MM-DD) |
| runtime | INTEGER | | 片长（分钟）|
| region | TEXT | | 地区/国家 |
| language | TEXT | | 语言 |
| rating | REAL | DEFAULT 0.0 CHECK(rating >= 0 AND rating <= 10) | 豆瓣评分 |
| rating_count | INTEGER | DEFAULT 0 | 评分人数 |
| review_count | INTEGER | DEFAULT 0 | 短评数 |
| box_office | REAL | DEFAULT 0 | 票房（万元）|
| ticket_price | REAL | DEFAULT 0 | 平均票价（元）|
| poster_url | TEXT | | 海报图片 URL |
| summary | TEXT | | 剧情简介 |
| douban_id | TEXT | | 豆瓣电影 ID（用于爬虫去重）|
| created_at | TEXT | DEFAULT (datetime('now','localtime')) | 记录创建时间 |
| updated_at | TEXT | DEFAULT (datetime('now','localtime')) | 记录更新时间 |

```sql
CREATE TABLE IF NOT EXISTS movies (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT NOT NULL UNIQUE,
    genre TEXT NOT NULL,
    director TEXT,
    actors TEXT,
    release_date TEXT,
    runtime INTEGER,
    region TEXT,
    language TEXT,
    rating REAL DEFAULT 0.0 CHECK(rating >= 0 AND rating <= 10),
    rating_count INTEGER DEFAULT 0,
    review_count INTEGER DEFAULT 0,
    box_office REAL DEFAULT 0,
    ticket_price REAL DEFAULT 0,
    poster_url TEXT,
    summary TEXT,
    douban_id TEXT,
    created_at TEXT DEFAULT (datetime('now','localtime')),
    updated_at TEXT DEFAULT (datetime('now','localtime'))
);
```

### 2.2 reviews — 短评表（可选）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |
| movie_id | INTEGER | NOT NULL REFERENCES movies(id) | 关联电影 ID |
| content | TEXT | NOT NULL | 短评内容 |
| rating | REAL | | 该短评的评分 |
| thumbs_up | INTEGER | DEFAULT 0 | 点赞数 |
| sentiment_score | REAL | | 情感分析得分 (-1.0 ~ 1.0) |
| created_at | TEXT | DEFAULT (datetime('now','localtime')) | 爬取时间 |

```sql
CREATE TABLE IF NOT EXISTS reviews (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    movie_id INTEGER NOT NULL REFERENCES movies(id) ON DELETE CASCADE,
    content TEXT NOT NULL,
    rating REAL,
    thumbs_up INTEGER DEFAULT 0,
    sentiment_score REAL,
    created_at TEXT DEFAULT (datetime('now','localtime'))
);

CREATE INDEX IF NOT EXISTS idx_reviews_movie_id ON reviews(movie_id);
```

### 2.3 box_office_log — 票房日志表（趋势图用）

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |
| movie_id | INTEGER | NOT NULL REFERENCES movies(id) | 关联电影 ID |
| date | TEXT | NOT NULL | 日期 (YYYY-MM-DD) |
| daily_box_office | REAL | DEFAULT 0 | 单日票房（万元）|
| total_box_office | REAL | DEFAULT 0 | 累计票房（万元）|

```sql
CREATE TABLE IF NOT EXISTS box_office_log (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    movie_id INTEGER NOT NULL REFERENCES movies(id) ON DELETE CASCADE,
    date TEXT NOT NULL,
    daily_box_office REAL DEFAULT 0,
    total_box_office REAL DEFAULT 0
);

CREATE INDEX IF NOT EXISTS idx_box_office_movie_date ON box_office_log(movie_id, date);
```

### 2.4 crawl_record — 爬虫记录表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |
| source | TEXT | NOT NULL | 数据来源（douban/maoyan）|
| status | TEXT | NOT NULL | 状态（success/failed/running）|
| records_count | INTEGER | DEFAULT 0 | 本次爬取记录数 |
| message | TEXT | | 错误信息/备注 |
| created_at | TEXT | DEFAULT (datetime('now','localtime')) | 爬取时间 |

```sql
CREATE TABLE IF NOT EXISTS crawl_record (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    source TEXT NOT NULL,
    status TEXT NOT NULL CHECK(status IN ('success', 'failed', 'running')),
    records_count INTEGER DEFAULT 0,
    message TEXT,
    created_at TEXT DEFAULT (datetime('now','localtime'))
);
```

### 2.5 system_config — 系统配置表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| key | TEXT | PRIMARY KEY | 配置键名 |
| value | TEXT | | 配置值 |
| description | TEXT | | 配置说明 |

```sql
CREATE TABLE IF NOT EXISTS system_config (
```

## 5. 数据初始化

### 5.1 首次启动流程

```
1. 检查数据库文件是否存在
   ├─ 不存在 → 创建数据库 + 建表 → 加载备份数据
   └─ 存在   → 检查版本 → 执行迁移（如有）
2. 检查数据是否为空
   ├─ 为空 → 尝试爬虫 → 爬虫失败 → 加载 CSV 备份
   └─ 有数据 → 直接使用
3. 更新系统配置
```

### 5.2 备份数据文件
- 路径：`data/backup_movies.csv`
- 格式：UTF-8 with BOM 编码
- 包含与 movies 表相同的字段
- 至少 50-200 条示例数据

## 6. 数据库操作规范

| 规范 | 说明 |
|------|------|
| **连接管理** | 使用单例模式管理连接，避免重复创建 |
| **事务处理** | 批量插入时使用事务（BEGIN/COMMIT）|
| **参数化查询** | 所有 SQL 使用 `?` 占位符，防 SQL 注入 |
| **资源释放** | 每次查询后关闭 cursor |
| **错误处理** | 所有数据库操作 try/except，记录日志 |
| **编码** | 统一使用 UTF-8 |

## 7. 数据库迁移策略

当表结构需要变更时：

```python
# db_manager.py 中维护版本迁移
MIGRATIONS = {
    '1.0': [
        # 初始版本，无需迁移
    ],
    '1.1': [
        "ALTER TABLE movies ADD COLUMN new_field TEXT;",
    ],
}

def check_migration():
    version = get_config('db_version')
    for ver, sqls in sorted(MIGRATIONS.items()):
        if ver > version:
            for sql in sqls:
                execute(sql)
            set_config('db_version', ver)
```

> 本项目 v1.0 暂不涉及迁移，预留接口供后续扩展。
