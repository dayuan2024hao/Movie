# 电影票分析系统 — 使用说明

## 1. 系统概述

电影票分析系统是一个 Windows 桌面端应用，综合票房、评分、评价、票价等因素进行电影数据可视化和智能推荐。

**主要功能：**

- 📊 **数据看板** — 9 种图表展示电影数据全局分析
- 🏆 **推荐榜** — 5 种算法智能推荐电影
- 🔍 **搜索** — 跨平台（豆瓣 + OMDB）实时搜索
- 📋 **电影详情** — 多源聚合展示电影完整信息
- ⚙️ **数据管理** — 从猫眼实时爬取 + CSV 导入

## 3. 快速启动

### 3.1 安装依赖

```bash
cd movie_analysis_system
pip install -r requirements.txt
```

### 3.2 配置 API Key（可选）

复制配置文件模板并填入你自己的 Key（如不配置，系统使用内置默认 Key 运行）：

```bash
copy config.example.py config.py
# 编辑 config.py，填入 OMDB API Key 和 TMDB API Key
```

- **OMDB API**：从 [www.omdbapi.com](https://www.omdbapi.com/) 免费注册获取
- **TMDB API**：从 [www.themoviedb.org/settings/api](https://www.themoviedb.org/settings/api) 注册获取

### 3.3 启动程序

```bash
python main.py
```

首次启动会自动：
1. 创建 SQLite 数据库（`data/movie_analysis.db`）
2. 从 CSV 备份数据加载电影
3. 尝试从猫眼爬取当前热映电影
4. 弹出主窗口

## 4. 界面导览

### 4.1 主窗口布局

```
┌─────────┬──────────────────────────────────────────────┐
│ 导航栏   │              内容区域                        │
│         │                                              │
│ 📊 看板  │   (根据所选页面切换内容)                      │
│ 🏆 推荐  │                                              │
│ 🔍 搜索  │                                              │
│ ℹ️ 关于  │                                              │
└─────────┴──────────────────────────────────────────────┘
```

- **导航栏**（左侧）：点击切换功能页面
- **内容区域**（右侧）：显示当前页面的具体内容

### 4.2 数据看板页面

看板包含 9 个图表和 5 个统计卡片：

| 区域 | 内容 | 说明 |
|------|------|------|
| 顶部工具栏 | 年份筛选 | 下拉选择年份范围，所有图表联动刷新 |
| 统计卡片行 | 5 项核心指标 | 总电影数、总票房、平均评分、平均票价、最高评分 |
| 图表面板 | 9 张分析图 | 票房 Top10、评分分布、类型占比、票房区间、票价分布、类型票房、年份趋势、四象限分析、散点图 |
| 底部 | 数据洞察 | 自动生成的文本分析总结 |
| 底部 | 电影数据表 | 所有电影的详细列表，可排序 |

> **鼠标滚轮**：在图表上滚动可正常翻页（已做事件转发）

### 4.3 推荐榜页面

5 种推荐算法，每种展示 Top20 电影：

| 算法 | 说明 | 适合场景 |
|------|------|----------|
| ⭐ 高分榜 | 按评分排序 | 找口碑最好的电影 |
| 🔥 热度榜 | 按评价人数排序 | 看大家都在关注什么 |
| 🏅 口碑榜 | 评分 × 评价人数综合 | 既有口碑又有热度 |
| 💰 价值榜 | 评分 ÷ 票价 | 找性价比最高的电影 |
| 🎯 综合榜 | 多维度加权评分 | 全面推荐的默认选项 |

点击「查看详情」可跳转到电影详情页。

### 4.4 搜索页面

- 输入电影名称，按回车或点击「搜索」按钮
- 系统并发搜索豆瓣和 OMDB 两个数据源
- 搜索结果自动去重
- 点击结果可直接打开电影详情页

### 4.5 电影详情页面

展示单部电影的完整信息：

| 区域 | 内容 |
|------|------|
| 海报 | 左侧 300×450px 高清海报（缓存机制） |
| 基本信息 | 标题、评分、票房、类型、地区、语言、片长 |
| 主演阵容 | 演员列表 |
| 剧情简介 | OMDB → 本地 → TMDB 三级降级获取 |
| 外部链接 | 特惠购票（热映电影）、豆瓣详情 |

## 5. 数据管理

### 5.1 自动爬取

每次启动程序时，系统自动：
1. 爬取猫眼热映电影列表
2. 对比数据库，标记已下映电影
3. 将新电影写入数据库

### 5.2 CSV 备份数据

`data/backup_movies.csv` 是初始数据备份文件，包含 86 部经典电影。

如需修改或补充数据：
1. 编辑 `data/backup_movies.csv`
2. 删除 `data/movie_analysis.db`
3. 重新运行 `python main.py` 自动重建数据库

### 5.3 手动刷新

重启程序即可触发重新爬取和数据更新。

## 6. 常见问题

### Q: 启动后没有数据？

首次启动会自动从 CSV 加载数据。如果网络不通，猫眼数据无法获取，系统会使用 CSV 中的已有数据正常展示。

### Q: 图表显示不全？

- 确保窗口最大化（推荐 1600×900 以上）
- 最小窗口尺寸为 1440×800

### Q: 搜索不到电影？

- 中文片名建议使用全名搜索
- 部分冷门电影可能 OMDB 没有收录
- 搜索结果来自豆瓣 suggest API + OMDB，网络问题可能导致搜索失败

### Q: 海报不显示？

- 豆瓣 CDN 对非浏览器请求返回 418，系统会自动使用渐变色兜底
- 海报下载后缓存到 `cache/posters/` 目录

### Q: 如何备份我的数据？

直接复制 `data/movie_analysis.db` 文件即可备份全部数据库内容。
