## 🔧 遇到的问题 & 解决方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| Windows 控制台 GBK 编码导致日志中文乱码 | Python logging StreamHandler 使用系统编码 | sys.stdout.reconfigure(encoding='utf-8') |
| check_and_migrate() 将版本从 1.0 提升到 1.1 | 迁移字典包含无 SQL 的 "1.1" 条目 | 增加 `if not sqls: continue` 跳过空迁移项 |
| 并发读写 `NoneType does not define __round__` | 同一 sqlite3.Connection 多线程竞争 | _write_lock → _db_lock，所有操作串行化 |
