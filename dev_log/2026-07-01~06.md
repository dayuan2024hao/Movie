## 🔧 关键问题 & 解决方案

### 1. 图表底部裁剪（v2.4b → v2.12 反复迭代 6 次）

**问题现象**：所有图表底部（X 轴标签、图例）被截断，看不完整。

**根因分析**：
- pyecharts InitOpts 默认 900×500px，容器实际只有 314~384px → 内部滚动条
- GridOpts(pos_bottom="30") 覆盖了 is_contain_label 的自动计算
- webview 高度(344px) < InitOpts 高度(364px) → 底部 20px 溢出
- 图例自身 25px + pos_bottom="5" = 30px 刚好被 overflow:hidden 切掉

**最终修复** (v2.12)：
1. InitOpts 全部设 width="100%" + 明确 height
2. GridOpts(is_contain_label=True) 删除所有 pos_bottom
3. webview 高度与 InitOpts 高度严格一致
4. CSS overflow:visible（替代 hidden）

### 2. 搜素功能卡死 UI（P0）

**问题**：搜索时界面完全卡死，无法操作。

**原因**：requests 请求在主线程同步执行，OMDB 超时（10s）堵塞 Qt 事件循环。

**修复**：新建 SearchWorker(QObject)，用 QThread + QMetaObject.invokeMethod(Qt.QueuedConnection) 跨线程安全更新 GUI。

### 3. 猫眼实时爬虫

**问题**：需要热映电影数据，但猫眼有反爬。

**修复**：改用桌面站 HTML (maoyan.com/films) + requests + 正则解析，不需要 Selenium 或浏览器。

### 4. OMDB 数据集成

**问题**：中文电影在 OMDB 搜不到。

**修复**：建立 EN_TITLE_MAP 中→英映射表（如"流浪地球2"→"The Wandering Earth 2"），搜不到时再用中文搜。加标题 word-match 校验防止返回不相关电影。

### 5. 数据库数据不完整

**问题**：130 条电影中 86 条没海报、56 条没票价、44 条没评价人数。

**早期修复** (v2.4b~v2.12)：
- 删除 CSV 中 24 条不存在的假电影（流浪地球3、哪吒3等）
- 移除爬虫 _mock_list() 硬编码假数据
- OMDB 批量补全：海报+简介（英文片名命中率约 30%）
- 按类型+年代估算票价
- 按评分估算评价人数
